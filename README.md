# Codex KLPT Demo Two

A second static HTML, Sass, and JavaScript demo that renders domains, optional subdomains, and elements from `data/domains.json`.

## Run It

Use a local server so the browser can fetch the JSON file. In VS Code, the Live Server extension is a good fit.

If you want to edit Sass:

```powershell
npm install
npm run sass:watch
```

The checked-in `dist/styles.css` is already compiled, so the page will run without installing packages.
